# 🎮 Stone Paper Scissor Game (Python)

## 📌 Description
This is a simple command-line based Stone, Paper, Scissor game built using Python.  
The user plays against the computer for a chosen number of rounds, and the final score determines the winner.

---

## 🚀 Features
- Play multiple rounds in one session  
- Random computer choices using Python’s `random` module  
- Input validation for user choices  
- Score tracking system  
- Option to replay the game  

---

## 🛠️ Tech Stack
- Python (Core concepts)
  - Loops (`while`, `for`)
  - Conditionals (`if-else`)
  - Dictionaries
  - Random module
